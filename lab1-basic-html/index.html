HTML Text Formatting

HTML provides various tags for text formatting:

  • Bold text using <b> or strong emphasis using <strong>
  • Italic text using <i> or emphasized text using <em>
  • Underlined text using <u>
  • Highlighted text using <mark>
  • Small text using <small>
  • Deleted text using <del>
  • Inserted text using <ins>
  • Subscript: H2O
  • Superscript: E=mc2
