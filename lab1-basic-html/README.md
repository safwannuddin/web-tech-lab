# Lab 1: Basic HTML

## Objective
Learn and implement basic HTML elements and text formatting.

## Topics Covered
- HTML document structure
- Headings (h1-h6)
- Paragraphs
- Text formatting tags (bold, italic, underline, etc.)
- Lists (ordered and unordered)
- Links and hyperlinks
- Blockquotes
- Horizontal rules

## Instructions
1. Open `index.html` in a web browser
2. Study the HTML code to understand different elements
3. Try modifying the content and adding your own elements

## Learning Outcomes
- Understand basic HTML document structure
- Use various text formatting elements
- Create lists and links
- Apply semantic HTML elements
