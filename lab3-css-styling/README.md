# Lab 3: CSS Styling and Layouts

## Objective
Master CSS styling techniques including typography, box model, flexbox, and grid layouts.

## Topics Covered
- CSS selectors and properties
- Typography and text styling
- Box model (margin, padding, border)
- Colors and backgrounds
- Flexbox layout
- CSS Grid layout
- Transitions and hover effects
- Responsive design with media queries

## Instructions
1. Open `index.html` in a web browser
2. Examine how CSS is linked to HTML
3. Study the `styles.css` file
4. Try modifying CSS properties to see effects
5. Resize the browser window to see responsive behavior

## Learning Outcomes
- Apply CSS styling to HTML elements
- Use flexbox for flexible layouts
- Create grid layouts with CSS Grid
- Implement responsive design
- Add interactive effects with transitions
