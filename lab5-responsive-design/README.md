# Lab 5: Responsive Web Design

## Objective
Learn to create responsive websites that adapt to different screen sizes using CSS media queries and flexible layouts.

## Topics Covered
- Viewport meta tag
- Fluid layouts with percentages
- CSS Grid for responsive layouts
- Flexbox for flexible components
- Media queries for different breakpoints
- Mobile-first design approach
- Responsive typography
- Responsive images
- Print styles

## Breakpoints Used
- **Mobile**: 480px and below
- **Tablet**: 481px to 768px
- **Desktop**: 769px to 1024px
- **Large Desktop**: 1025px and above

## Instructions
1. Open `index.html` in a web browser
2. Resize your browser window to see responsive behavior
3. Test on different devices if available
4. Use browser DevTools to simulate different screen sizes
5. Study the media queries in `responsive.css`

## Learning Outcomes
- Implement responsive design principles
- Use media queries effectively
- Create flexible layouts with CSS Grid and Flexbox
- Design mobile-first websites
- Optimize user experience across devices
