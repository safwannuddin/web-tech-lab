# Lab 6: Interactive JavaScript Features

## Objective
Build interactive web components using JavaScript event handling and DOM manipulation.

## Topics Covered
- Event handlers (onclick, onkeypress, onmouseover, etc.)
- Image gallery with lightbox
- To-do list with add/delete/complete functionality
- Accordion menu
- Tab navigation system
- Modal dialogs
- Interactive counter
- Dynamic content creation
- CSS class manipulation
- Event delegation

## Interactive Components
1. **Image Gallery**: Click images to view in lightbox, navigate with arrows
2. **To-Do List**: Add tasks, mark as complete, delete tasks
3. **Accordion**: Click to expand/collapse sections
4. **Tabs**: Switch between different content panels
5. **Modal**: Open and close modal dialogs
6. **Counter**: Increment, decrement, and reset counter

## Instructions
1. Open `index.html` in a web browser
2. Interact with each component
3. Open browser console (F12) to see any messages
4. Study `interactive.js` to understand event handling
5. Modify the code to add your own features

## Learning Outcomes
- Handle various JavaScript events
- Create interactive UI components
- Manipulate DOM elements dynamically
- Implement common UI patterns
- Add and remove CSS classes with JavaScript
- Build reusable interactive components
