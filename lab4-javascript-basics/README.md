# Lab 4: JavaScript Basics and DOM Manipulation

## Objective
Learn JavaScript fundamentals and how to manipulate HTML elements using JavaScript.

## Topics Covered
- Variables and data types (string, number, boolean, array, object)
- Functions and parameters
- Arithmetic operations
- Conditional statements (if-else, switch)
- Loops (for loop)
- Array operations
- DOM manipulation
- Event handling
- Getting and setting element values

## Instructions
1. Open `index.html` in a web browser
2. Open browser console (F12) to see console messages
3. Interact with each section to see JavaScript in action
4. Study `script.js` to understand the code
5. Try modifying functions to experiment

## Learning Outcomes
- Understand JavaScript syntax and basic concepts
- Write functions to perform operations
- Manipulate HTML elements using JavaScript
- Handle user events
- Work with different data types
- Use conditional logic and loops
