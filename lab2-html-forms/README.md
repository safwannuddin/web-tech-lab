# Lab 2: HTML Forms

## Objective
Learn to create interactive HTML forms with various input types and form elements.

## Topics Covered
- Form structure
- Text input fields
- Email and telephone inputs
- Date input
- Radio buttons
- Checkboxes
- Select dropdown
- Textarea
- Password field
- Form validation attributes
- Submit and reset buttons

## Instructions
1. Open `index.html` in a web browser
2. Interact with different form elements
3. Try submitting the form to see validation
4. Experiment with adding new form fields

## Learning Outcomes
- Create functional HTML forms
- Use different input types appropriately
- Implement basic form validation
- Understand form attributes and methods
