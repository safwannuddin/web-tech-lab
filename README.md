# Web Technology Lab Programs

A comprehensive collection of web development laboratory programs covering HTML, CSS, and JavaScript fundamentals.

## Lab Programs Overview

### Lab 1: Basic HTML
- **Topics**: HTML structure, text formatting, lists, links, semantic elements
- **Folder**: `lab1-basic-html/`
- **Files**: `index.html`, `README.md`

### Lab 2: HTML Forms
- **Topics**: Form elements, input types, validation, radio buttons, checkboxes
- **Folder**: `lab2-html-forms/`
- **Files**: `index.html`, `README.md`

### Lab 3: CSS Styling and Layouts
- **Topics**: CSS selectors, box model, flexbox, grid, transitions, responsive design
- **Folder**: `lab3-css-styling/`
- **Files**: `index.html`, `styles.css`, `README.md`

### Lab 4: JavaScript Basics and DOM Manipulation
- **Topics**: Variables, functions, arrays, conditionals, loops, DOM manipulation
- **Folder**: `lab4-javascript-basics/`
- **Files**: `index.html`, `script.js`, `README.md`

### Lab 5: Responsive Web Design
- **Topics**: Media queries, mobile-first design, flexible layouts, breakpoints
- **Folder**: `lab5-responsive-design/`
- **Files**: `index.html`, `responsive.css`, `README.md`

### Lab 6: Interactive JavaScript Features
- **Topics**: Event handling, lightbox, to-do list, accordion, tabs, modals
- **Folder**: `lab6-interactive-javascript/`
- **Files**: `index.html`, `interactive.css`, `interactive.js`, `README.md`

## Getting Started

1. Clone this repository
2. Navigate to any lab folder
3. Open the `index.html` file in a web browser
4. Read the lab-specific `README.md` for instructions and learning objectives

## Prerequisites

- Basic understanding of web technologies (helpful but not required)
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A text editor or IDE for viewing/editing code

## Learning Path

It's recommended to complete the labs in order:
1. Start with Lab 1 to understand HTML basics
2. Progress to Lab 2 for forms
3. Learn styling in Lab 3
4. Add interactivity in Lab 4
5. Make it responsive in Lab 5
6. Build advanced features in Lab 6
